# Mini Social Post Application

## About Project

This is a simple social media application where users can:

* Sign Up and Log In
* Create Posts
* Upload Images
* Like Posts
* Comment on Posts
* View Posts from All Users

## Technologies Used

* React.js
* Node.js
* Express.js
* MongoDB Atlas
* Material UI

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```
